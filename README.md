# Farm Management PWA

An offline-first Progressive Web App for dairy and poultry farmers. This app helps farmers manage their livestock operations, track production, monitor health records, manage finances, and generate professional reports.

### Core Features
- **Offline-First**: Works completely offline with local data storage
- **PWA Ready**: Installable on devices with app-like experience
- **Dual Livestock Support**: Manage both dairy and poultry operations
- **Cloud Sync**: Optional Firebase sync for data backup and multi-device access
- **Rule-Based Insights**: Smart insights without AI dependencies

### Production Management
- Track milk/egg production
- Monitor herd/flock size and status
- Record daily production metrics
- View production trends with charts

### Financial Management
- Record income and expenses
- Track feed, veterinary, and operational costs
- Monitor profit margins
- Generate financial reports

### Health Management
- Record health treatments and vaccinations
- Schedule upcoming health events
- Track animal health history
- Set reminders for vaccinations

### Reporting
- Generate professional PDF reports
- Export data to CSV
- Production summaries
- Financial reports
- Health overviews
